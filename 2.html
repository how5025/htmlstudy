2. 웹과 앱부터 만들어라

https://www.youtube.com/watch?v=wpUiN5hBnyc

전반적인 코딩 공부를 하기 위해서는 먼저 가장 쉽고, 흥미 있는 웹개발 부터 배우는 것이 좋다.
웹개발은 - Front - End (Clint side) : 보이는 화면 부분. 쉽고 재미있음.
Back - End (Server side) : 정보를 저장, 처리, 가공
깊게 배우려 하지 말고 필요한 것부터 습득하면서 윤곽을 알아가는 것이 효과적.

가. Front - End : html, css, java script 학습
html - 연필과 같음. 웹의 밑그림, 뼈대 그리는 역할
css - 물감과 같음. 밑그림에 색칠, 수정
javascript - 그림을 움직이게 만드는 역할. 웹의 요소를 선택하고 제어할 수 있을 정도만 간단히 공부하고 넘어감
프론트앤드만 배워도 구글, 네이버등의 API를 이용해서 상당히 쓸만한 이용이 가능하다.
##API : 미리 만들어 놓은 가져다 쓸 수 있는 기능.

나. Back - End - 초기부터 깊게 배우려하지 말고, 필요한 것 그때그때 찾아서 공부할 것.
javascript, ruby, python등 쉬운 언어 이용해서 가장 기본적인 개념인
변수, 조건문, 반복문, 함수 4가지 개념만 학습 추천.
백엔드는 원래 알아야할 것이 많은 분야였다.(SQL, Database, Session, Cookie등등) 그러나
지금은 웹프레임워크의 도움을 받아 개발할 수 있다.
프레임워크 종류 : java는 spring, python은 django, ruby는 rails등의 프레임워크 활용.
초보자에게는 ruby on rails. 가장 적게 배우고 가장 많은 일을 할 수 있었다
.
다. 이렇게 웹을 만들게 되면 앱도 만들 수 있다. 또한 게임도 비슷한 원리로 만들 수 있다.
