3.코딩공부 셋팅

https://www.youtube.com/watch?v=p_v_js0mxVc

프로그래밍 언어는 대표적으로 5가지 언어가 있다.
C,C++ / JAVA / Python / html, css, java script / php 위 언어 중 어떤 것을 먼저 배워야 한다는 질문은 맞는 질문이 아니다. 무엇을 만들 것이냐에 따라서 배워야 할 언어가 다르기 때문이다. 따라서 어떤 것을 먼저 배워야 하느냐가 아니라, 어떤 것을 만들 것인 지를 먼저 정해야 한다.

1> C, C++ : 기계 등 하드웨어에 적합한 언어.(에어컨, 스마트 티비, 냉장고등). 속도가 빠르고 가벼워 메모리가 클 필요가 없는 것에 사용됨. 또는 photshop, premier pro등 소프트웨어 퍼포먼스가 필요한 큰 기업에서 사용. 2> java - 정부, 은행, 기업용 웹어플리케이션 빌드에 사용. 안드로이드 어플 만들 수 있음. Front-End 개발등에 유리. 3> Python : 초보자들을 위한 가장 좋은 선택 => 다양한 용도로 사용 가능. 컴퓨터해킹, 머신러닝, 데이터사이언스, 데이터 비쥬얼라이제이션, 웹스크래핑 웹사이트빌딩에 사용. 읽기 편하고 쉬운 언어임. 4> Html, CSS, Java script : 웹사이트를 만들기 위한 언어. 웹개발자가 되려면 선택해야 함. 5> php : Back - End 개발. 대부분 웹은 word press 때문에 php에서 작동함.
